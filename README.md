### Compte-rendu TP1 Introduction GIT

Dans ce TP on apprend à travailler avec git, mais avant tout il faut le configurer nous pouvons le faire a l'aide des commandes : 

--La commande **git config user.name  "nom"**  qui est la pour modifier le nom configuré sur git.
--La commande **git config user.email "mail"** qui est la pour modifier le mail configuré sur git.

ce qui nous permet donc de pouvoir commencer a travaillier avec git.

puis nous avons vue dans la suite de ce tp les commandes de base permettant d'utilisé git voici si dessous les commandes que nous avons appris :  </br>

--La commande **git commit -m fichier "commentaire"** permet d'inclure dans le dépot git un fichier et de déposez si besoin un commentaire lors de cette ajout. </br>
--La commande **git add** permet quant a elle de selectionnez les fichiers qui pourront par la suite être inclue dans le dépot git. </br>
--La commande **git status** permet de voir l'état actuelle des modifications entre le local et le depot. </br>
--La commabde **git log** permet de vérifier tout les modifications du dépots git. </br>
